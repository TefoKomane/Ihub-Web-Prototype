Web Development Stream

FormatFull-time, in person
Duration6 months

Participants learn HTML, CSS, JavaScript, responsive design, modern front-end frameworks, and API integration through hands-on projects for local partners.

Training is paired with communication coaching, teamwork practice, and professional habits so graduates are ready to join entry-level tech roles or build freelance projects with confidence.

HTML CSS JavaScript Responsive Design Frameworks API Integration
Ideal for Aspiring front-end developers, junior software engineers, UX builders and learners who want to turn practical skills into opportunity.

The programme is offered at no cost and is focused on learners who want hands-on experience, mentorship and career support in the digital economy.
